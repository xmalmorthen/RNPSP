Catalogos de Marcas
# Descripcion URL Tipo Paramentros Entrada
1 Obtiene el listado completo de marcas
  • api/catalogos/marcas
 GET
2 Obtiene los datos de una sola marca
  • api/catalogos/marca
 GET
  • id
3 Agrega una marca en el catalogo
  • api/catalogos/addMarca
 POST
  • descipcion
  • activo (opcional)(default = 1)
4 Actualiza la descripción de una marca en el catalogo
  • api/catalogos/updateMarca
 POST
  • id (id_Marca)
  • descripcion
5 Actualiza el campo activo de una marca en el catalogo
  • api/catalogos/updateActivoMarca
 POST
  • id (id_Marca)
  • activo
6 Obtiene el listado de activos del catalogo de marcas
/api/catalogos/marcasActivos

$this->load->model('CaMarcas_model');
$this->CaMarcas_model->getActivos(); 		GET
7 Busqueda por descripción del catalogo de marcas
/api/catalogos/busquedaMarcas

$this->load->model('CaMarcas_model');
$this->CaMarcas_model->busquedaMarcas({term}); 		POST
  • term (descripcion a buscar)
  • activo
