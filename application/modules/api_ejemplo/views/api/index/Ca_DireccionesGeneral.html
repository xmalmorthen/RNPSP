Catalogo de Direccion General
# Descripcion URL Tipo Paramentros Entrada
1 Obtiene el listado completo de direciones generales URL completo:

URL sin path:
api/catalogos/direccionesGeneral
Llamar desde controlador:
$this->load->model('api/CaDireccionesGeneral_model');
$this->CaDireccionesGeneral_model->get_idDependencia({id_Dependencia},{activo}); 		GET
  • id(id_Dependencia)
  • activo (opcional) (1: trae solo los activos)
2 Obtiene el listado completo de activos de direciones generales URL completo:

URL sin path:
api/catalogos/direccionesGeneralActivos
Llamar desde controlador:
$this->load->model('api/CaDireccionesGeneral_model');
$this->CaDireccionesGeneral_model->get_idDependencia({id_Dependencia},true); 		GET
  • id(id_Dependencia)
