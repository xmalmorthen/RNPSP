# Descripcion URL Tipo Paramentros Entrada
1 Obtiene el listado de autorizadores por idDireccionArea URL completo:
URL sin path:
api/catalogos/autorizadores
Llamar desde controlador:
$this->load->model('api/DeAutorizadores_model');
$dependencias = $this->DeAutorizadores_model->getUsuarios_idDireccionArea( {idDireccionArea} ); 		GET
  • id( {idDireccionArea} )
2 Agrega un autorizador URL completo:
URL sin path:
api/catalogos/autorizador
Llamar desde controlador:
$this->load->model('api/DeAutorizadores_model');
$dependencias = $this->DeAutorizadores_model->set_Autorizador({idUsuario},{idDireccionArea}); 		POST
  • idUsuario
  • idDireccionArea
3 Elimina un autorizador URL completo:
URL sin path:
api/catalogos/autorizador
Llamar desde controlador:
$this->load->model('api/DeAutorizadores_model');
$dependencias = $this->DeAutorizadores_model->delete_Autorizador({idUsuario},{idDireccionArea}); 		DELETE
  • idUsuario
  • idDireccionArea
