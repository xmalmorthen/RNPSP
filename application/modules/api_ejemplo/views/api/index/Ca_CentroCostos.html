Catalogo de Centro de Costos
# Descripcion URL Tipo Paramentros Entrada
1 Obtiene el listado completo de centro de costos URL completo:
URL sin path:
api/catalogos/centroCostos
Llamar desde controlador:
$this->load->model('api/CaCentroCostos_model');
$dependencias = $this->CaCentroCostos_model->get(); 		GET
2 Obtiene el detalle de un centro de costos URL completo:
URL sin path:
api/catalogos/centroCosto
Llamar desde controlador:
$this->load->model('api/CaCentroCostos_model');
$dependencias = $this->CaCentroCostos_model->getCentroCosto( {ID CENTRO COSTO} ); 		GET
  • id({ID CENTRO COSTO})
